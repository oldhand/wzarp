
wzarpp（Windows+Tezancloud+Apache+Redis+PostreSQL+PHP）是一个功能强大的集成软件包。

目前，简化配置与安装，下载即用的原则。

注意事项：
   1，全绿色版本
   2，只能安装在任意盘符的raid5目录下
   3, 请先安装当前目录下的软件狗驱动，以及微软运行库
   4，启动只需要执行raid5目录下的run.bat文件
   5，本地访问请访问http://127.0.0.1:7000
   6, 内网访问请访问http://您的IP:7000 
   7, 请保证软件狗插在USB接口上
   8，请赋予apps目录everyone的所有控制权，才能保存图片等的正常上传。
   9，当出现异常问题，可以直接运行run.bat文件，将自动清除异常状态。
